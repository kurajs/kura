# Kura example: docs

A first-version **documentation site MVP** on June — the Mintlify/Fumadocs browsing
experience, with the "humans **and** agents" surface built in (not bolted on).

```sh
npm run gen   # freeze content/docs/*.md -> app/_content.ts
npm run dev   # http://localhost:3000
```

## What it has

**For humans (the browsing experience):**
- 3-column layout: **sidebar** nav tree (grouped by `section`, ordered by `order`),
  rendered content, right-hand **ToC** with heading anchors.
- **Semantic search** (`/search`, press `/` to focus) over the docs via `@kurajs`.
- **Copy-as-Markdown** + **View-as-Markdown** + **Open in ChatGPT / Claude** buttons.
- Per-code-block copy, breadcrumbs, prev/next pager.

**For agents (nearly free on June — the differentiator):**
- Per-page raw Markdown: append `.md` to any URL (`/docs/search.md`).
- `/.json` projection, `/llms.txt` index — auto-generated by June.
- `/mcp` MCP server with `search_docs` (semantic) + `get_page` tools.

One source, four surfaces (HTML · Markdown · JSON · MCP), zero drift.

## How it maps to Mintlify / Fumadocs

| feature | Mintlify | Fumadocs | this MVP |
|---|---|---|---|
| sidebar tree | `docs.json` | file tree + `meta.json` | frontmatter `section`/`order` |
| ToC from headings | ✓ | ✓ | ✓ (ids injected, anchors) |
| search | hosted | Orama | **semantic (bge-m3)** |
| copy page as markdown | ✓ | `<LLMCopyButton>` | ✓ (fetches `.md`) |
| per-page `.md` | ✓ | route handler | **free (June projection)** |
| `llms.txt` | ✓ | `llms()` | **free (June)** |
| MCP server | ✓ (paid) | ✗ | ✓ (`defineAction`) |

## Structure

```
content/docs/*.md        frontmatter: title, description, section, order
app/_content.ts          frozen by `june gen`
app/_nav.ts              sidebar groups, prev/next, ToC extraction
app/_kb.ts               semantic search index (@kurajs + bge-m3)
app/_view.tsx            DocsShell (chrome) + DocsPage (content + actions)
app/page.tsx             home (= introduction)
app/docs/[slug]/page.tsx doc route (+ .md / .json projections)
app/search/page.tsx      search results
app/actions.ts           search_docs + get_page MCP tools
app/global.css           the theme
```

Run `june gen` after editing content.
