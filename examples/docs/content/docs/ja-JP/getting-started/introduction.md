# はじめに

多くのドキュメントツールは Markdown を HTML に変換してそこで終わります。エージェントは
その HTML を再びテキストへとスクレイピングする —— 情報が失われる往復です。

Kura はソースを保持します。すべてのページは、読者が見る HTML としても、あなたが書いた
ままの Markdown としても提供されます。さらに JSON 投影と MCP ツール面も。ソースは一つ、
ずれは生じません。

蔵（くら）という名は「倉庫」を意味します —— 知識を蓄え、常に最新に保つ場所です。
