# セマンティック検索

Kura は各ページを小さな多言語モデル（既定では bge-m3）で埋め込み、凍結された
インデックスにベクトルを保存します。クエリは一度だけ埋め込まれ、コサイン類似度で
ランク付けされます —— だから「どうやって支払いを受けるか」という問いが、その語句を
含まないページでも請求ページに辿り着きます。

インデックスはバンドルに同梱されます。Cloudflare Workers ではコンテンツを変えずに、
ローカルの埋め込みを Workers AI へ差し替えられます。
