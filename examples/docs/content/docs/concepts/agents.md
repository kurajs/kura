# Agent-native

A Kura site serves humans and agents from the same source. Humans get HTML; agents get
Markdown, JSON, and MCP — the same words, in the form each reader needs.

This is what "agent-native" means here: not a chatbot bolted onto the docs, but a
knowledgebase whose every page is legible to a model by construction.
