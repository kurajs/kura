# Introduction

Most documentation tools render Markdown to HTML and stop there. Agents are then
left to scrape that HTML back into text — a lossy round-trip.

Kura keeps the source. Every page is served as the HTML a reader sees **and** as the
exact Markdown you wrote, plus a JSON projection and an MCP tool surface. One source,
no drift.

The name 蔵 (*kura*) means "storehouse" — a place where knowledge is kept and kept
evergreen.
