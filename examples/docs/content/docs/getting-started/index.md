# Get started

Kura is a documentation framework built on [June](https://june.build). You author
plain Markdown; Kura projects it four ways — HTML for readers, Markdown and JSON for
tooling, and MCP for agents — all from the same source.

This section gets you from an empty folder to a running docs site.

- **[Introduction](/docs/getting-started/introduction)** — what Kura is and who it is for.
- **[Quickstart](/docs/getting-started/quickstart)** — a knowledgebase in three steps.
