# Projections

A projection is a view of the same source. Write a page once and Kura serves it as:

- **[Markdown](/docs/features/projections/markdown)** — the source, verbatim, at `.md`.
- **[JSON](/docs/features/projections/json)** — structured metadata + body at `.json`.
- **[MCP](/docs/features/projections/mcp)** — a tool surface agents can call.

The HTML you are reading is just the fourth projection.
