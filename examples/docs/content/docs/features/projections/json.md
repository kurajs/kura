# JSON

Append `.json` to any doc URL for a structured view: `slug`, `title`, `section`,
`locale`, the raw `markdown`, and the parsed `body`.

It is the projection to reach for when you are building tooling on top of a Kura site —
a changelog generator, a link checker, or a custom search UI.
