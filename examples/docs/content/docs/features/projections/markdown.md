# Markdown

Append `.md` to any doc URL and Kura returns the file you wrote, frontmatter included.
This page lives at `/docs/features/projections/markdown.md`.

Because it is the source — not HTML converted back to text — an agent reads exactly
what the author intended, links and code fences intact.
