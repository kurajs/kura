# Overview

Kura ships the pieces a knowledgebase needs, and nothing it doesn't.

- **[Search](/docs/features/search)** — semantic by default, with a lexical fallback
  that runs anywhere.
- **[Projections](/docs/features/projections)** — one Markdown source, served as HTML,
  Markdown, JSON, and MCP.
