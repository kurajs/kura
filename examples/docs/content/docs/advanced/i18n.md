# Internationalization

Kura localizes by mirroring. Put a locale's pages under `content/docs/<locale>/`,
matching the default folder layout:

```
content/docs/features/search/semantic.md        # default (en)
content/docs/ja-JP/features/search/semantic.md  # Japanese
```

Translate only what you need — a missing page falls back to the default language, with a
notice on the page. Folder titles localize too: drop a `meta.json` in the mirrored
folder and its `title` overrides the default for that locale.
