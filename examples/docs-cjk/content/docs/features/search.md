# 搜尋

這個文件站的搜尋是純關鍵字(BM25),不需要任何模型即可運作。

## 中文斷詞

查詢「向量搜尋」會被切成詞,再比對文件。本頁示範的詞彙包含:軟體開發、網路通訊、
程式設計、滑鼠操作、以及資料庫索引。(關鍵字不加粗體——`**` 等格式標記在斷詞時
會把詞切開,而 `stripMdx` 並不會移除它們。)

## 繁简通用

繁體與简体的用詞不同(軟體 vs 软件、網路 vs 网络、程式 vs 程序)。本範例在索引與
查詢時都先用 OpenCC 把文字折算成同一套繁體用詞,所以即使讀者輸入简体的「软件」
「网络」「程序」,也能命中這份繁體文件。
