# Introduction

**PROJECT_NAME** is a knowledgebase for humans and agents, built on
[June](https://june.build) with [Kura](https://kura.build). Humans get a fast docs site
with sidebar, table of contents, and search; AI agents get the same content as Markdown
and as callable MCP tools — one source, zero drift.

## One source, four surfaces

Every page is served as:

- **HTML** for humans (this page),
- **Markdown** at the same URL with `.md` appended (try `introduction.md`),
- **JSON** with `.json`,
- and an **MCP tool** (`search_docs`) at `/mcp` that agents can call.
